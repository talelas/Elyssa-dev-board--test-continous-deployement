# Elyssa Dev Board

ESP32-S3FN8 based development board.

## Features

- ESP32-S3FN8
- 8 MB Flash
- USB Programming
- Dedicated Gyroscope I2C
- SD Card SPI
- Primary SPI
- PWM support
- Arduino IDE support

---

## Installation

Open Arduino IDE

File → Preferences

Additional Board Manager URLs:

https://bejaouihamza.github.io/elyssa/package_bejaouihamza_elyssa_index.json

Open:

Tools → Board → Boards Manager

Search:

Elyssa

Install.

Select:

Tools → Board → Elyssa Dev Board

Done.
